# Calculator – Simple & Fast Arithmetic Tool

A lightweight and responsive calculator built using HTML, CSS, and JavaScript.
Yeh calculator basic arithmetic operations perform karta hai — jaise addition, subtraction, multiplication, division — aur percentage (%) functionality bhi support karta hai.
Clean UI, smooth button clicks, aur real-time expression evaluation isko beginner-friendly aur easy-to-understand banate hain.
